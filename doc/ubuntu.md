# Ubuntu VM

- 04 cores
- 12 GB RAM
- 2 GB RAM VGA (Dùng chế độ chia sẻ GPU, sẽ cấu hình chế độ chia sẻ GPU khi có card GPU)
- 50 GB ổ cứng

## Version

Ubuntu Server 24.04

## Purpose

- Bộ phần mềm LibreOffice (tích hợp sẵn trong Ubuntu)
- Firefox hoặc tương đương dùng cho duyệt Web
- VLC dùng xem file video
- Bộ gõ tiếng Việt
- ThunderBird (dùng tải email chế độ offline SMTP hoặc IMAP)

## Services

SSH
